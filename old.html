SayUmm Hackathon Entry

SayUmm is a virtual reality environment using Google Cardboard and Unity which allows the user to practice giving presentations and speeches. Voice data is recorded using IBM BlueMix API which is then analyzed in a Python library and parsed to a dynamic webpage. Placed as a Top Ten Hack at PennApps XII.

RunOrFund Hackathon Entry

Run or fund is a dynamic web application that promotes charitable giving whilst encouraging physical wellness. Firebase is used for OAuth and data storage. Won Best use of Amazon Web Services at HackRPI II.

Defined Personal Project

Open Source online dictionary that uses the WolframAlpha API to allow the user to define multiple words at once, uses AJAX and JQuery to handle requests.
